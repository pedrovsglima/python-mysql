Integrando recursos do MySQL e do Python¶

Objetivo¶

Criar um projeto onde seja possvel utilizar ambos os recursos do MySQL e da linguagem Python.

Base de dados¶

Esta base de dados consiste em notas obtidas por alunos em vrias disciplinas. Nela esto inclusas pontuaes em trs exames e uma variedade de fatores pessoais, sociais e econmicos que podem ter efeitos de interao sobre eles.

Atributos:

  • gender: sexo biolgico ('female' ou 'male');
  • race/ethnicity: etnia ('group A', 'group B', 'group C', 'group D' ou 'group E');
  • parental level of education: escolaridade dos pais ('bachelor's degree', 'some college', 'master's degree', 'associate's degree', 'high school' ou 'some high school');
  • lunch: tipo de almoo ('standard' ou 'free/reduced');
  • test preparation course: curso de preparao para o teste ('none' ou 'completed');
  • math score: nota no teste de matemtica (valor inteiro de 0 a 100);
  • reading score: nota no teste de leitura (valor inteiro de 0 a 100);
  • writing score: nota no teste de escrita (valor inteiro de 0 a 100).

Esta uma base de dados fictcia e deve ser usada apenas para fins educacionais.

Disponvel em: https://www.kaggle.com/spscientist/students-performance-in-exams

Análises¶

A ideia deste documento mostrar as funcionalidades das duas ferramentas trabalhando juntas. Para efeito de aprendizado, sero levantadas algumas questes entendidas como relevantes e em seguida sero apresentados possveis grficos que podem contribuir na resposta de cada uma delas, seguidos de uma breve discusso.

Questes levantadas:

  1. Qual a mdia e desvio padro das notas dos 3 exames de acordo com a escolaridade dos pais dos alunos?

  2. Considerando a mdia de aprovao como 70, quantos alunos foram aprovados em todas disciplinas? Destes, qual a porcentagem de quem fez curso preparatrio?

  3. O tipo de almoo do aluno pode ser considerado um fator que contribui para os resultados dos testes?

2. Alterar o tipo de dados de algum atributo visando otimização¶

1. Qual a média e desvio padrão das notas dos 3 exames de acordo com a escolaridade dos pais dos alunos?¶
In [8]:
# definir query
query = """SELECT parental_level_of_education, 
        AVG(math_score), AVG(reading_score), AVG(writing_score),
        STD(math_score), STD(reading_score), STD(writing_score)
        FROM studentsperformance
        GROUP BY parental_level_of_education
        ORDER BY parental_level_of_education"""


# executar comando e armazenar resultado
result = pd.read_sql(query, con)


# para usar o parmetro yerr, se faz necessrio uma modificao na estrutura do data frame
# mais detalhes sobre:
# https://stackoverflow.com/questions/23144784/plotting-error-bars-on-grouped-bars-in-pandas
# https://stackoverflow.com/questions/23000418/adding-error-bars-to-grouped-bar-plot-in-panda
yerr = result[['STD(math_score)','STD(reading_score)','STD(writing_score)']].to_numpy().T


# visualizar resultado
result.plot(kind = 'bar', 
            x = "parental_level_of_education", 
            y = ["AVG(math_score)", "AVG(reading_score)", "AVG(writing_score)"],
            yerr = yerr)


# configuraes do plot
plt.title("Mdia e desvio padro das notas dos alunos de acordo com a escolaridade dos seus pais")
plt.xlabel("Escolaridade dos pais")
plt.xticks(rotation=30, horizontalalignment="center")
plt.ylabel("Nota")
plt.legend(bbox_to_anchor=(1, 1))

plt.show()

Discussão - Questão 1¶

As mdias das notas dos alunos com os pais de maior grau de escolaridade (masters degree) esto prximas dos outros alunos, sendo um pouco maior do que aqueles de pais com high school e um pouco menor do que aqueles de pais com bachelors degree. Ou seja, um maior grau de escolaridade dos pais do aluno no necessariamente implica em uma melhor nota nos exames.

Por outro lado, os alunos com os pais de menor grau de escolaridade (some high school) so os que apresentam menor mdia nos trs exames. Qualquer afirmao em cima dessa informao necessita de verificao por anlise estatstica, alm de consideraes de influncia de outros fatores, exemplo: sociais e econmicos.

2. Considerando a média de aprovação como 70, quantos alunos foram aprovados em todas disciplinas? Destes, qual a porcentagem de quem fez curso preparatório?¶

# definir nota de aprovao
media = 70


# computar total de registros
total = pd.read_sql("""SELECT COUNT(*) FROM studentsperformance""", con) 
total = total.iloc[0][0]


# definir query - alunos aprovados nos 3 exames, de acordo com o curso preparatrio
query = """SELECT test_preparation_course, COUNT(*)
        FROM studentsperformance
        WHERE math_score>= {} AND reading_score>= {} AND writing_score>= {}
        GROUP BY test_preparation_course
        ORDER BY test_preparation_course
        """.format(media, media, media)


# executar comando e armazenar resultado
result = pd.read_sql(query, con)


# extrair total de alunos que se aplicam na condio
total_fez_curso = result['COUNT(*)'].sum()

Discussão - Questão 2¶

Do total, 26% dos alunos foram aprovados nos trs exames. Desses, apenas 42% fizeram o curso preparatrio. Ou seja, de modo geral, a realizao de um curso preparatrio no um bom fator para relacionar aprovao dos exames, j que a minoria daqueles aprovados fizeram o curso.
